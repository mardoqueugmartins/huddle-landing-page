#  Huddle Landing Page — Responsive Front-End Implementation

Landing page responsiva desenvolvida a partir de um desafio de layout, com foco em estrutura semântica, fidelidade visual, organização de CSS e responsividade real entre desktop e mobile.

Este projeto demonstra domínio de fundamentos sólidos de front-end sem uso de frameworks.

#### Frontend Mentor challenges
---

##  Project Overview

A aplicação consiste em uma landing page institucional de seção única, estruturada para apresentar:

- Proposta de valor clara
- Hierarquia tipográfica bem definida
- Layout moderno
- Call to Action destacado
- Ícones sociais com estados interativos

O desenvolvimento priorizou qualidade estrutural, organização visual e experiência consistente entre diferentes tamanhos de tela.

---

##  Objectives

- Implementar layout fiel ao design original
- Aplicar boas práticas de HTML5 semântico
- Utilizar CSS Grid e Flexbox de forma estratégica
- Garantir responsividade real (desktop → mobile)
- Manter código limpo e organizado
- Trabalhar estados de interação (hover)

---

##  Tech Stack

- HTML5 (estrutura semântica)
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- Google Fonts (Poppins & Open Sans)
- Font Awesome

Sem uso de frameworks ou bibliotecas externas de layout.

---

##  Technical Decisions

### 1. Layout Structure

A estrutura principal foi construída utilizando **CSS Grid** para organizar:

- Header
- Conteúdo principal
- Footer

Isso garante controle previsível do layout e melhor organização estrutural.

---

### 2. Content Alignment

O conteúdo principal utiliza **Flexbox** para:

- Alinhamento horizontal no desktop
- Reorganização vertical no mobile
- Controle de espaçamento entre elementos

---

### 3. Responsive Strategy

A responsividade foi aplicada com:

- Media queries específicas para breakpoint mobile
- Mudança de background entre desktop e mobile
- Reorganização da estrutura de layout
- Ajustes tipográficos
- Adaptação de espaçamentos

---

### 4. Interactive States

- Botões possuem estado `hover`
- Ícones sociais possuem feedback visual
- Transições suaves melhoram percepção de qualidade
